sailor.assetcentral.utils.parse_filter_parameters(equality_filters=None, extended_filters=(), property_mapping=None)[source]¶

Parse equality and extended filters into breakable and unbreakable filters.

The distinction between breakable and unbreakable filters is necessary because the AssetCentral endpoints are generally queried via GET requests with filter parameters passed in the header, and there is a limit to the header size. This allows breaking a query into multiple queries of shorter length while still retrieving the minimal result set the filters requested.
